# HYDROS Commit Message Guide

Use these exact messages when uploading each batch of files:

## Initial Upload
```
Initial commit: Add HYDROS README with complete system overview, specs, and reference documentation
```

## Documentation Files
```
Add PSR package, technical manual, and calculations (docs/)
```

## CAD Files
```
Add AutoCAD master file and engineering drawings (cad/)

Note: HYDROS_MASTER.dwg is a large binary file (CAD format). 
If file size exceeds GitHub limits, consider using Git LFS (Large File Storage).
```

## Simulation Results
```
Add SimScale CFD and FEA simulation outputs (simulation/)
```

## Calculations
```
Add computation sheet and supporting calculations (calculations/)
```

## Design Iterations
```
Add design iteration log and appendices
```

## Final Submission
```
Final submission: HYDROS PSR complete May 2026
```
